## 合同风险点标注

### 1. 技术服务合同
**合同类型**: SERVICE
**主要风险点**:
- 🔴 **高**: 预付款比例较高（30%），存在甲方资金风险
- 🟡 **中**: 分期支付，存在拖欠尾款风险
- 🟡 **中**: 交付延期条款违约金仅0.5%/天，可能不足以弥补损失
- 🟢 **低**: 保密期仅3年，相对较短
- 🟡 **中**: 知识产权条款约定乙方保证不侵权，但无明确赔偿责任

### 2. 保密协议
**合同类型**: NDA
**主要风险点**:
- 🟡 **中**: 竞业限制未约定具体补偿金额，可能无效
- 🔴 **高**: 违约金100万元可能过高，存在被法院调整风险
- 🟢 **低**: 保密期限2年较为合理
- 🟢 **低**: 保密范围定义明确

### 3. 劳动合同
**合同类型**: EMPLOYMENT
**主要风险点**:
- 🟡 **中**: 未明确试用期工资是否为80%的约定工资
- 🟡 **中**: 加班工资计算方式需验证合法性
- 🔴 **高**: 未约定竞业限制补偿金，竞业限制条款可能无效
- 🟢 **低**: 合同解除条件相对合理
- 🟡 **中**: 知识产权归属甲方，符合职务作品规定

### 4. 销售合同
**合同类型**: SALES
**主要风险点**:
- 🔴 **高**: 预付款30%比例较高，甲方资金风险大
- 🟡 **中**: 交付期30天较紧，存在延期风险
- 🟡 **中**: 质保期36个月较长，乙方承担较大风险
- 🟢 **低**: 违约金设有5%上限，但对甲方保护有限

## 使用方式

### 在系统中测试

1. 进入"合同上传"页面：http://localhost:5173/new-task

2. 填写合同信息：
   - 合同名称：根据实际文件名填写
   - 对方当事人：根据合同内容填写
   - 合同类型：选择对应类型（SERVICE/SALES/NDA/EMPLOYMENT）

3. 上传文件：
   - 点击或拖拽文件到上传区域
   - 系统会自动解析并进入分析流程

4. 创建知识库集合：
   - 进入"知识库管理"：http://localhost:5173/kb
   - 创建对应的知识库集合（如"合同法知识库"、"劳动法知识库"等）
   - 上传相关法律法规文档

5. 查看分析结果：
   - 等待分析完成（约2-3分钟）
   - 查看"Results"页面获取风险列表
   - 使用"Review"页面进行人工审核

- 中风险：分期支付风险、知识产权侵权风险

- 中风险：交付期紧张、质保期长
- 低风险：违约金相对合理

## 重新生成

如需重新生成 DOCX 和 PDF 文件：

```bash
cd test-data/contracts
python3 generate_contracts.py
```

如需修改合同内容：

1. 编辑对应的 TXT 文件
2. 重新运行生成脚本
3. 新的 DOCX/PDF 文件会覆盖旧文件
